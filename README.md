## Install

- download onnxruntime gpu from https://elinux.org/Jetson_Zoo#ONNX_Runtime
- i am using pythin 3.8
- onnxruntime 1.12.1
- then install from local wheel file

  ```
  pip install onnxruntime_gpu-1.12.1-cp38-cp38-linux_aarch64.whl
  ```
